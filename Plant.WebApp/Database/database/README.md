#  Base de Datos

Este proyecto usa **SQL Server**.

##  Pasos para preparar la base de datos
1. Abre **SQL Server Management Studio (SSMS)** o **Azure Data Studio**.
2. Conéctate a tu instancia de SQL Server.
3. Ejecuta primero el script [`01_create_schema.sql`] para crear la tabla.
4. Después ejecuta el script [`02_seed_data.sql`] para insertar datos de ejemplo.

Al terminar tendrás la tabla `Plantas` con registros de prueba listos para usar.
